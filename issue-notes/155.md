# これまでの課題：現在updateサブコマンドがバグっている

- log：
```
現在のプロセスの終了を待っています
cargo installを起動しています
$ cargo install --force --git https://github.com/cat2151/mascot-render-server
Updating git repository `https://github.com/cat2151/mascot-render-server`
error: multiple packages with binaries found: mascot-render-server, psd-viewer-tui. When installing a git repository, cargo will always search the entire repo for any Cargo.toml.
Please specify a package, e.g. `cargo install --git https://github.com/cat2151/mascot-render-server mascot-render-server`.
更新に失敗しました。終了コード: 101
Enterキーを押すと閉じます
```

# 対策：
- 以下のcommandが生成されること

```
cargo install --force --git https://github.com/cat2151/mascot-render-server mascot-render-server psd-viewer-tui

```
# 方法：
- まずcat-self-update-libのAPI仕様を調査し、上記が可能か不可能かを調査して報告すること
- 調査できないなら失敗を報告すること
- ベンダリング等の「その場しのぎの実装に逃げる」ことを禁止する
