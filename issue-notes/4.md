# issue TUIにお気に入り機能を実装する。fでお気に入り登録。vでお気に入りlistからENTERで選ぶ。削除はtoml直接編集 #4
[issues #4](https://github.com/cat2151/mascot-render-server/issues/4)

- layer listにて、fでお気に入り登録
- vでお気に入りlistを表示、jkと上下とpageUpDownで選んでENTERで決定
- お気に入りlistは、cache/ ではなく favolites/ に、toml保存、で検証
- 削除はtoml直接編集（小さく始める。しばらく検証してから削除をlistから可能かつbackupから復元なども検討）
