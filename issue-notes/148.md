# issue server_skin.logに、現在の実際の表示pngファイル名を出力する #148
[issues #148](https://github.com/cat2151/mascot-render-server/issues/148)

- これまでの課題：
    - 現在、REST POST requestで四国めたんskinへの変更をrequestしたとき、server.logには成功と出ているが、表示されているskinはあんこもんのまま、といったバグが続いており、
    - agent側、アプリ側は、バグ原因を特定できず、憶測で「修正しました」という回答ばかりしており、現実が見えていない
- 対策：
    - 描画の最終段において、実際の現実のpngのskin情報をlogに出すことで、現実をagentにみせる、つまり可視化をする
